# 🚀 2024 Vibe Coding 期末專案：[心理測驗_✨你最像哪種小動物?✨ㄒ
## 🌟 1. 專題願景 (Vibe)
* **目標對象：** 任意年齡
* **設計風格：** 極簡可愛風
* **核心功能：**簡單的小測驗，希望能在其中得到快樂
## 🤖 2. Gemini 協作紀錄 (Prompt Log)
*請貼出你餵給 Gemini 最關鍵的一個 Prompt：*你現在是一名心理師，你能根據一些問答看出符合什麼樣的結果。
> **範例指令：** 「請當我的心理師，我想要做一個心理測驗的網站
以下是我的題目:
測驗主題
✨你最像哪種小動物？✨

（結果類型示例：🐶小狗、🐱小貓、🦊狐狸、🐰兔子）
題目設計（8 題，單選）
1️⃣ 假日你最常做什麼？
A. 和朋友出去玩
B. 一個人追劇或滑手機
C. 做點有挑戰性的事
D. 睡到自然醒
2️⃣ 在朋友眼中，你比較像？
A. 很好相處、很暖
B. 有點神秘、不太愛說話
C. 很聰明、點子多
D. 安靜可愛型
3️⃣ 遇到不熟的人，你通常會？
A. 主動聊天
B. 先觀察一下
C. 用幽默化解尷尬
D. 有點緊張，但還是會微笑
4️⃣ 你最喜歡的天氣是？
A. 晴天
B. 下雨天
C. 涼爽微風
D. 冬天窩在室內
5️⃣ 如果今天心情不好，你會？
A. 找人聊天
B. 自己消化情緒
C. 做點別的事轉移注意力
D. 吃點好吃的
6️⃣ 在團體中，你通常是？
A. 氣氛製造者
B. 默默聽大家說話的人
C. 出主意的人
D. 跟著大家就好
7️⃣ 你比較嚮往哪種生活？
A. 熱鬧、有很多朋友
B. 安靜、自在
C. 有挑戰、很自由
D. 穩定、舒服
8️⃣ 選一個你最喜歡的詞
A. 陪伴
B. 自由
C. 聰明
D. 溫柔
（給你用的）結果對應概念
A 多 → 🐶 小狗型：熱情、重視朋友
B 多 → 🐱 小貓型：獨立、慢熱
C 多 → 🦊 狐狸型：聰明、有想法
D 多 → 🐰 兔子型：溫柔、可愛
(幫你想測驗結果的可愛文案)
我想要偏向「燕麥色」的背景，字體優雅的襯線體，按鈕要大圓角
且最後結果可以放對應動物的可愛圖片」
*在開發過程中，你遇到了什麼 Bug？Gemini 怎麼幫你修好的？*
* **遇到的問題：** 圖片一直不對應
* **Gemini 的建議：** 我說:幫我對作中結果生成相對應的圖片(大概說了三次才稍微成功)
* * **遇到的問題：**在手機上格式跑掉
* **Gemini 的建議：** 我說:請幫我稍微修改，讓手機格式正確
## 📂 3. 檔案結構說明
* `index.html`：網頁主結構
* `style.css`：視覺美化 (如果有的話)
* `assets/`：存放圖片或素材的資料夾
